Baremetal & FreeRTOS

从 SDK V1.1.3 开始，项目中引入了 FreeRTOS，用户可以在项目中选择使用 BareMetal 或 FreeRTOS。默认配置为使用 BareMetal。

编译器开关设置在文件：APP_CompileOption.h

#define APP_FREERTOS_ENABLE APP_FALSE

如果宏 APP_FREERTOS_ENABLE 设置为 APP_TRUE ，FreeRTOS 调度器将用于调度 TaskHandler 函数。

如果宏 APP_FREERTOS_ENABLE 设置为 APP_FALSE ，将使用一个简单的 while 循环来运行 TaskHandler 函数。

Since SDK V1.1.3 FreeRTOS is introduced into the project and allows user to choose between BareMetal and FreeRTOS within the project. The default configuration is set to use BareMetal.

The compiler switch is set in file: APP_CompileOption.h

#define APP_FREERTOS_ENABLE APP_FALSE

If the macro APP_FREERTOS_ENABLE is set to APP_TRUE, FreeRTOS scheduler will be used to schedule TaskHandler function.

If the macro APP_FREERTOS_ENABLE is set to APP_FALSE, a simple while loop will be used to run TaskHandler function.

SDK High Level Design

Tip

During initialization, the UART module interacts with the Chipsbank Library (CB_LIB) to establish communication. This step ensures that the system is prepared for data transmission and reception.

At runtime, the User interacts with the system by sending commands and arguments through the CB_LIB. Upon receiving data, the CB_LIB triggers an interrupt callback (IRQ), prompting the Uart_Commander module in the System Application to come into action. The Uart_Commander parses the received command and associated arguments, preparing them for further processing.

Depending on the nature of the command, the Uart_Commander triggers the execution of UWB applications (uwbApps). These applications utilize the UWB API provided by the CB_LIB to carry out tasks related to Ultra-Wideband (UWB) communication.

在初始化期间，UART模块与Chipsbank Library（CB_LIB）交互以建立通信。这一步确保系统准备好进行数据传输和接收。

在运行时，用户通过CB_LIB发送命令和参数与系统交互。接收到数据后，CB_LIB触发中断回调（IRQ），促使系统应用中的Uart_Commander模块开始运行。Uart_Commander解析接收到的命令和相关参数，并为进一步处理做准备。

根据命令的性质，Uart_Commander触发执行UWB应用程序（uwbApps）。这些应用程序利用由CB_LIB提供的UWB API执行与超宽带（UWB）通信相关的任务。

UART Commander

SDK框架提供了直观的演示和控制，允许用户使用诸如Teraterm之类的串行终端工具发出命令，指示固件执行专用操作，例如UWB TX、RX、PDOA和TSTP。

UART 0 的默认 GPIO 配置是 GPIO0 用于 RX ， GPIO1 用于 TX 。

默认配置请参照下方的 Tera Term 截图：



为了获得更好的用户体验，请进行以下配置：

  • New line (Receive): LF

  • Local echo: Enabled



在配置完Tera Term之后，您应该能够在启动CB5000V210时看到打印输出，使用SDK。

  • CB LIB version: X.X.X

The SDK framework facilitates intuitive demonstration and control, allowing users to issue commands using serial terminal tools such as Teraterm to instruct the firmware to perform dedicated operations, such as UWB TX, RX, PDOA, and TSTP.

The default GPIO configuration for UART 0 is GPIO0 for RX and GPIO1 for TX.

For default configuration, please refer to the screenshot of Tera Term below:



For a better user experience, please configure the following as well:

  • New line (Receive): LF

  • Local echo: Enabled



After configuring Tera Term, you should be able to see a printout when powering up CB5000V210 with SDK.

  • CB LIB version: X.X.X

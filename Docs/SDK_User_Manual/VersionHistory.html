Version History

V2.0.0

New Features

  1. [SDK] SDK Restructure

V1.4.11

New Features

  1. [UWB] RF FLow 2.5: Using new logic of setBits functions, fix packet format(SP1)

  2. [UWB] DSTWR: Using absolute timer

V1.4.10

New Features

  1. [UWB] Implement frequency with 50Hz (single sided AOA) (Update driver, flow optimization, reduce printing time, log format for single sided rngaoa)

  2. [UWB] Add the leading edge index for every RX chain after PDOA

  3. [UWB] Open the access 255 CIR points after PDOA

  4. [UWB] Bring the COMM TX time down to ~200us

V1.4.9

New Features

  1. [CPU] Optimize UART IRQ to speedup processing

  2. [CPU] Release SCR, Sleep, DeepSleep library as source code.

  3. [CPU] Revise RC compensation to ensure QSPI Flash Tvsl more than 300us.

  4. [CPU] Bug fixed on Wdt module

  5. [CPU] Fix I2C driver polling mode bug

  6. [CPU] Set GPIO default value as 0, 1

  7. [CPU] Set default antenna ID as demo_3d (CB_AOA_ANTENNA_0_1)

  8. [UWB] Fix Periodic TRX RX port number to 0,1,2 in user command argument.

  9. [UWB] Fix PSR printout disabled.

  10. [UWB] Fix APP COMM-TRX time stamp use wrong API.

  11. [UWB] Fix APP COMM-TX take longer time (~380 us)

  12. [UWB] Default as coexist mode, instead of sensitivity mode

  13. [UWB] Power code default as 28

  14. [UWB] RNGAOA: Improve SYNC mechanism with ACK frame

  15. [UWB] RNGAOA: Simply framework - remove second sync

  16. [UWB] RNGAOA: Bugfix - clean TRX dataram to avoid message misprocessed as SYNC frame.

V1.4.8

New Features

  1. [UWB] Refine self adaptive model for AOA for Antenna ID EN_ANTENNA_2_0_3D_A

  2. [UWB] Data log format change in RNGAOA APP, Fills in -10000 if nothing

  3. [CPU] Enable HW Floating Point

V1.4.7

New Features

  1. [UWB] Added self adaptive model for AOA

V1.4.6

New Features

  1. [UWB] Configuration of parameters through function call or s command

  2. [UWB] Option to perform Double-sided PDOA in APP-UWB-RNGAOA

  3. [UWB] Able to fetch PDOA CIR results

Bug Fixes & Minor Changes

  1. UART commands a,b,c,d,e,f functions have the configurable parameters removed
